# Markdown Task Manager

A simple script to track tasks using Node.js, pnpm, and TypeScript.

## Setup

1. Ensure you have Node.js installed (version 18 or higher recommended).
2. Install pnpm globally: `npm install -g pnpm`
3. Install dependencies: `pnpm install`

## Usage

- Build the project: `pnpm run build`
- Run the script: `pnpm start`
- Run in development mode: `pnpm run dev`
- Lint the code: `pnpm run lint`
- Fix linting issues: `pnpm run lint:fix`
- Format the code: `pnpm run format`

## Troubleshooting

- If you encounter issues with TypeScript, ensure it's installed: `pnpm add -D typescript`
- For Node.js version issues, check your Node version with `node --version`
